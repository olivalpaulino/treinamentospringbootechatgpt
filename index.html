Aprenda a criar rapidamente APIs REST com o framework mais popular do Java. Com nosso treinamento, você aprenderá a integrar com facilidade diferentes serviços, como bancos de dados, e serviços de mensagens, para criar uma API REST altamente funcional e eficiente.


Neste treinamento, você aprenderá Spring Web com Spring Data JPA, utilizando MySQL, como banco de dados relacional, fazendo o mapeamento objeto relacional (ORM), Spring Initializr para acelerar a criação do projeto de API Rest com o gerenciados de dependências do Maven, utilizando as versões 3.1.0 e 2.7 do Spring Boot, versões 8 e 17 do Java, versão 8.0.32 do MySQL. Um treinamento completo e atualizado.


O treinamento também inclui aulas práticas de preparação de ambiente de desenvolvimento utilizando o sistema operacional Windows, instalação do kit de desenvolvimento do java (Java JDK 8 e 17), instalação do Eclipse como IDE principal, adicionando a extensão do Spring Boot Framework 4, instalando também o MySQL, Postman para testar a API Rest.

Conteúdo do Treinamento

O Treinamento é composto por 4 módulos, sendo o primeiro módulo de preparação do ambiente e boas-vindas, o segundo módulo de criação de projetos, onde o aluno já começa a por a mão na prática, para se habituar a utilizar as ferramentas. No terceiro módulo é explicado em detalhes a criação de uma API Rest e o quarto e último módulo, você aprenderá mais sobre o Spring Web, Spring Data JPA com MySQL, Maven e a Arquitetura Rest API.


Também será apresentado a criação de requisições web via protocolo http, mapeando rotas/endpoints com Spring Web, entendo o passo a passo da construção da API, adaptando de acordo com a sua necessidade.


Para finalizar, entenderemos mais sobre Consumo de API com a visão do back-end e front-end, tendo como ferramenta potencializadora do aprendizado, o ChatGPT, os códigos das aulas disponibilizados no Github, e um instrutor para tirar dúvidas.



Demais Informações

O treinamento possui aulas inteiramente gravadas, em alta qualidade, para que o aluno assista de acordo com a sua necessidade, horário e local.


As aulas possuem duração de 15 a 30 minutos, com exceção dos projetos práticos, que podem varia de 1 hora a 2 horas, pois envolvem tanto a parte prática, quanto a parte conceitual. Dessa forma, o aprendizado é acelerado e já tiramos o maior número de dúvidas comuns que possam surgir.
