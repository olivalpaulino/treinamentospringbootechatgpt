Conteúdo do Treinamento

O Treinamento é composto por 4 módulos, sendo que o primeiro módulo é composto pela mensagem de boas-vindas, e a preparação do ambiente de desenvolvimento, com o download, instalação e configuração das ferramentas de desenvolvimento.

O segundo módulo é composto pelas aulas de criação de projetos, onde o aluno já começa a por a mão na prática, para se habituar a utilizar as ferramentas, de forma simples e didática.

Já no terceiro módulo é explicado em detalhes, o processo de criação de uma API, com detalhes sobre a arquitetura, utilização e desenvolvimento.

No quarto módulo, você aprenderá mais sobre as dependências Spring Web, Spring Data JPA, Spring DevTools, em conjunto com o banco de dados relacional, MySQL, o gerenciador de dependências Maven, e o consumo de API via Postman.

Também será apresentado a criação de requisições web via protocolo http, mapeando os endpoints, com a dependência Spring Web, entendo o passo a passo dessa construção, sem a necessidade de fazer uma imersão em redes de computadores.

Para finalizar, entenderemos mais sobre Consumo de API com a visão do back-end e front-end, tendo como ferramenta potencializadora do aprendizado, o ChatGPT, os códigos das aulas disponibilizados no Github, e um instrutor para tirar dúvidas.
